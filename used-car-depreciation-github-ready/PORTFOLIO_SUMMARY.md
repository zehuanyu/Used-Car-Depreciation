# Portfolio Summary

## Project Title

Used Car Depreciation Analysis and Price Prediction

## One-Sentence Pitch

An end-to-end data science project that analyzes the drivers of used-car depreciation and builds a machine learning pipeline to estimate resale value from listing-level features.

## Highlights

- Built a structured workflow covering cleaning, feature engineering, EDA, modeling, benchmarking, error analysis, and app-based presentation.
- Achieved strong historical predictive performance, with prior full-run results around $1.7k MAE and 0.97 R2.
- Added segment-level analysis to identify where the model is reliable and where it needs improvement.
- Packaged the work into a GitHub-ready repository with sample data, documentation, and a Streamlit demo.
